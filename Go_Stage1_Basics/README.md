# Go Stage1 Basics

Go 语言基础阶段学习内容，含章节源码、实验、知识点文档、项目实践等。

## 目录结构

| 目录                | 说明                                                         |
| :------------------ | :----------------------------------------------------------- |
| `src/`              | 章节源码，子目录对应核心章节（如`chapter1_基础语法`），含`chapterX.go`（代码示例）、`chapterX.md`（知识点梳理），为`lab/`实验提供基础参考 |
| `lab/`              | 实验，子目录对应章节实验（如`lab1_基础语法`），含`labX.md`（实验要求 + 步骤）、`labX.go / taskX.go / projectX.go`（实验代码），巩固章节知识点 |
| `docs/`             | 知识点文档，按编码、包机制、值类型等拆分的 Markdown 文档，辅助理解`src/`和完成`lab/`实验 |
| `projects/`         | 项目实践，存放基础阶段小型项目，综合运用知识点，提升工程化能力 |
| `Go_Primer_Book.md` | 按章节系统梳理 Go 基础知识的参考书，适合快速查阅遗忘点或作为复习资料 |
| `Go_ProjectArchitecture_And_NamingConventions.md` |项目架构与命名规范文档，指导 Go 项目的工程化结构设计与代码命名规则 |
