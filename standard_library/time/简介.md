# Go 标准库 `time` 包 

**定位**：后端开发**最常用**标准库之一，处理时间、日期、时间戳、定时、休眠等所有时间相关操作

**适用场景**：留言创建时间、日志时间戳、接口超时、定时任务、时间格式化（项目必用）

**风格**：新手友好、全实战、可永久留存

------

## 一、包基础信息

### 1. 是什么？

`time` 是 Go 官方**标准库**，专门用于**时间的获取、格式化、计算、转换、休眠**等操作。

### 2. 核心作用

- 获取当前系统时间
- 时间格式化（给用户展示好看的时间）
- 时间戳转换（存储到数据库 / 内存）
- 时间加减 / 比较
- 程序休眠、定时执行
- 日志 / 业务记录时间（服务器黑匣子必备）

### 3. 导入方式

```
import "time"  // 开箱即用，无需下载
```

------

## 二、必备基础：时间单位常量（直接用，不用手写数字）

`time` 包封装了常用时间单位，**全是常量，直接调用**：

```
time.Second      // 1 秒
time.Minute      // 1 分钟
time.Hour        // 1 小时
time.Millisecond // 1 毫秒
time.Hour * 24   // 1 天（官方无天常量，用小时拼接）
```

------

## 三、核心功能大全（后端开发必用，附实战代码）

### 功能 1：获取当前时间（最基础）

获取系统当前的时间对象 `time.Time`

```
// 获取当前本地时间
now := time.Now()

// 输出示例：2026-03-25 16:30:45.123456 +0800 CST
fmt.Println(now)
```

------

### 功能 2：时间格式化（**最重要！页面展示必用**）

Go 语言格式化有**唯一固定参考时间**：

`2006-01-02 15:04:05`（Go 语言诞生时间，死记这个字符串）

#### 常用格式化模板（直接复制用）

```
now := time.Now()

// 1. 年-月-日
now.Format("2006-01-02") // 2026-03-25

// 2. 年-月-日 时:分:秒（最常用，你的留言板展示时间用这个）
now.Format("2006-01-02 15:04:05") // 2026-03-25 16:30:45

// 3. 只保留时分秒
now.Format("15:04:05")

// 4. 中文格式
now.Format("2006年01月02日 15时04分05秒")
```

------

### 功能 3：时间戳（存储 / 传输用，纯数字）

时间戳 = 从 1970 年到现在的秒数，**占用空间小，方便存储**

```
now := time.Now()

// 1. 获取秒级时间戳（最常用）
timestamp := now.Unix() // 1742968245

// 2. 毫秒级时间戳
now.UnixMilli()
```

#### 时间戳 → 时间对象（反向转换）

```
// 把时间戳转回 time.Time
t := time.Unix(1742968245, 0)
```

------

### 功能 4：时间计算（加减、比较）

适用于：过期时间、创建时间距今多久、判断时间先后

```
now := time.Now()

// 1. 时间加法：当前时间 + 1小时
later := now.Add(time.Hour)

// 2. 时间减法：当前时间 - 1天
before := now.Add(-time.Hour * 24)

// 3. 比较两个时间（返回bool）
now.After(before)  // true：现在 是否在 之前 之后
now.Before(later)  // true：现在 是否在 之后 之前
```

------

### 功能 5：程序休眠（暂停执行）

适用于：接口限流、模拟等待、定时

```
// 休眠 1 秒
time.Sleep(time.Second)

// 休眠 500 毫秒
time.Sleep(time.Millisecond * 500)

// 休眠 1 分钟
time.Sleep(time.Minute)
```

------

### 功能 6：获取时间的单独部分

拆分年、月、日、时、分、秒

```
now := time.Now()

now.Year()   // 2026
now.Month()  // March
now.Day()    // 25
now.Hour()   // 16
now.Minute() // 30
now.Second() // 45
```

## 四、留言板项目实战（直接复用）

### 场景 1：给留言添加创建时间

```
// store.Message 结构体中添加时间字段
type Message struct {
    Nickname  string
    Content   string
    CreatedAt time.Time // 留言创建时间
}

// 创建留言时自动记录当前时间
func (s *MemoryStore) Add(nickname, content string) {
    s.mu.Lock()
    defer s.mu.Unlock()
    msg := Message{
        Nickname:  nickname,
        Content:   content,
        CreatedAt: time.Now(), // 自动记录创建时间
    }
    s.messages = append(s.messages, msg)
}
```

### 场景 2：HTML 页面格式化展示时间

```
<!-- 前端模板 -->
{{ .CreatedAt.Format "2006-01-02 15:04:05" }}
```

### 场景 3：日志打印带时间（配合 log 包）

```
log.Printf("创建时间：%s，新留言来自：%s", now.Format("2006-01-02 15:04:05"), nickname)
```

## 五、`time` 包高频使用总结（背会这 5 个就够工作用）

1. `time.Now()` → 获取当前时间
2. `Format("2006-01-02 15:04:05")` → 格式化展示
3. `Unix()` → 获取秒级时间戳
4. `Add()` → 时间加减
5. `time.Sleep()` → 程序休眠

------

## 六、新手必避坑（关键！）

1. **格式化必须用固定参考时间**：`2006-01-02 15:04:05`，不能写别的时间！
2. **时区默认是本地时区**，国内直接用没问题
3. `time.Time` 是结构体，**可以直接存储**，不用转字符串
4. 时间戳是数字，**适合存储**；格式化字符串适合展示

------

## 七、终极总结

`time` 包是后端开发**离不开**的工具：

- 记录业务时间（留言 / 订单 / 日志）
- 格式化时间给用户看
- 时间戳用于存储传输
- 休眠 / 定时控制程序流程
