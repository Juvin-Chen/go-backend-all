# Go Web 编程：最简单的 Go Web 应用程序

## 核心说明

Go 语言仅靠标准库即可构建 Web 应用程序，无需第三方依赖，相比其他语言更简洁。

## 完整可运行代码

```go
package main

import "net/http"

func demo1() {
	// 第一个参数是路由地址，/ 为根地址
	// 注册匿名处理器函数到 DefaultServeMux 的根路径
	http.HandleFunc("/", func(w http.ResponseWriter, r *http.Request) {
		w.Write([]byte("hello world")) // 向客户端返回 "hello world" 响应
	})

	// 启动 Web 服务器
	// 参数1：监听地址（localhost:8080）；参数2：处理器（nil 表示使用默认的 DefaultServeMux）
	http.ListenAndServe("localhost:8080", nil) // default servemux
}

// 程序入口
func main() {
	demo1()
}
```

## 关键要点解析

1. `http.HandleFunc("/", 匿名函数)`：将匿名处理器函数注册到默认路由 `DefaultServeMux` 的根路径 `/`，用户访问根路径时触发该函数；
2. `http.ListenAndServe("localhost:8080", nil)`：启动 HTTP 服务，监听 `localhost:8080` 端口；第二个参数为 `nil`，表示使用 `DefaultServeMux` 作为默认处理器；
3. `w.Write([]byte("hello world"))`：通过 `ResponseWriter` 向客户端返回文本响应。


## 总结

1. 核心逻辑：注册处理器函数到默认路由 → 启动 HTTP 服务 → 处理并返回响应；
2. 关键特性：`ListenAndServe` 第二个参数为 `nil` 时，自动使用 `DefaultServeMux` 作为路由分发器。
